## 🚨 Troubleshooting

### Problem: Server won't start
**Solution:** Check DEPLOYMENT.md

### Problem: Database error
**Solution:** Delete ecommerce.db and restart

### Problem: Products not showing
**Solution:** Verify http://localhost:3000/api/products

### Problem: Login not working
**Solution:** Create new account or check browser console

### Problem: Can't add to cart
**Solution:** Make sure you're logged in first

## 🌐 API Endpoints

### Authentication
- POST /api/auth/register
- POST /api/auth/login

### Products
- GET /api/products
- GET /api/categories

### User
- GET /api/users/:id
- PUT /api/users/:id

### Cart
- GET /api/cart/:userId
- POST /api/cart
- DELETE /api/cart/:cartId

### Orders
- POST /api/orders
- GET /api/orders/:userId
